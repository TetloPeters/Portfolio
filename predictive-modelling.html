Project Overview

This project focused on developing predictive models to solve real-world problems in healthcare and education. Using Python machine learning libraries, I built models to predict prostate cancer outcomes from patient data and automated grading systems for student performance evaluation.

The goal was to demonstrate how predictive analytics can transform healthcare decision-making and improve efficiency in academic assessment processes.

Key Models Developed

Prostate Cancer Outcome Prediction

  • Applied classification algorithms (Logistic Regression, Random Forest, XGBoost)
  • Used patient clinical data including PSA levels, tumor stage, and biopsy results
  • Achieved accuracy of 87% with Random Forest model
  • Enabled early detection support for healthcare providers

Automated Student Grading System

  • Developed ML pipeline to grade short text responses using NLP
  • Used bag-of-words and TF-IDF for feature extraction
  • Compared predictions against human grading benchmarks
  • Reduced grading time while maintaining fairness and accuracy
