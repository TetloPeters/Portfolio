Project Overview

This comprehensive analytics project combined crime trend analysis, sales forecasting, and property valuation into an integrated data intelligence solution. By leveraging SQL for data extraction, Python for advanced analytics, and visualization tools for stakeholder communication, the project delivered actionable insights across multiple domains.

The project addressed real-world challenges in public safety, real estate investment, and business planning through sophisticated data cleaning, trend analysis, and predictive modeling techniques.

Crime Analysis Component

Data Sources & Processing

Integrated crime data from multiple law enforcement databases, standardizing incident reports, location data, and temporal information to create a unified analytical framework.

Key Insights Delivered

  • Crime Hotspots: Identified 800 N Alameda as the highest-frequency location with 2,297 incidents
  • Temporal Patterns: Peak crime occurrence between 12:00-18:00, with significant decline from 00:00-06:00
  • Crime Classification: 59% classified as less serious offenses, 41% as serious crimes
  • Weapon Analysis: N/A weapons (641,475 cases) dominated, indicating non-violent crimes; unknown weapons (35,682) and verbal threats (23,766) followed
  • Premises Distribution: Streets (248,363 incidents) were most common, followed by single-family dwellings (160,622)

Crime Type Breakdown

  • Vehicle theft: 107,266 incidents (highest volume)
  • Battery/Assault: 74,626 incidents
  • Burglary from vehicle: 60,178 incidents
  • Theft of identity: 59,801 incidents
  • Vandalism/Felony: 56,701 incidents
Predictive Analysis: Developed time-series forecasting models to predict crime trends, enabling law enforcement agencies to optimize resource allocation and patrol scheduling during high-risk periods.

Data Cleaning & Preparation

Challenges Addressed

  • Missing Data: Handled null values in weapon descriptions, location coordinates, and incident classifications
  • Inconsistent Formatting: Standardized date/time formats, address conventions, and crime category labels
  • Duplicate Records: Identified and removed duplicate incident reports using composite key matching
  • Outlier Detection: Flagged and investigated anomalous entries for data quality assurance
  • Categorical Encoding: Transformed text-based categories into numerical representations for analytical processing

SQL Data Extraction

Utilized complex SQL queries to join multiple crime databases, filter by date ranges, aggregate incident counts by location and type, and calculate temporal statistics. Optimized query performance through indexing and partitioning strategies for large datasets.

Python Analytics Pipeline

Implemented Python-based ETL workflows using pandas for data manipulation, NumPy for numerical computations, and scikit-learn for predictive modeling. Automated data cleaning processes reduced manual intervention by 80% and ensured consistent data quality across all analytical outputs.

Sales Forecasting Component

Methodology

Applied time-series analysis techniques including moving averages, exponential smoothing, and ARIMA models to forecast future sales based on historical patterns, seasonal trends, and external market factors.

Key Features

  • Trend Analysis: Identified long-term growth patterns and cyclical variations in sales data
  • Seasonality Detection: Quantified seasonal impact on sales performance across different product categories
  • Demand Forecasting: Generated 3-month, 6-month, and 12-month sales projections with confidence intervals
  • Scenario Planning: Modeled optimistic, realistic, and pessimistic scenarios for strategic planning
  • Alert System: Implemented automated alerts for significant deviations from forecasted trends
Business Impact: Sales forecasting accuracy improved by 23%, enabling better inventory management, resource allocation, and revenue planning. Reduced stockouts by 35% and overstock situations by 28%.

Property Valuation Component

Valuation Model

Developed a comprehensive property valuation system integrating multiple data sources including historical sales records, property characteristics, neighborhood crime statistics, market trends, and economic indicators.

Analytical Features

  • Comparative Market Analysis: Automated comparison of subject properties against comparable sales
  • Location Intelligence: Integrated crime data to assess neighborhood safety impact on property values
  • Price Trend Analysis: Historical price movements and appreciation rates by area
  • Investment ROI Calculator: Projected returns based on rental income potential and market appreciation
  • Risk Assessment: Evaluated investment risks considering crime rates, market volatility, and economic factors

Data Integration

  • Property listing databases (MLS data)
  • Crime statistics (law enforcement databases)
  • Demographics and economic indicators
  • School ratings and amenities
  • Transportation accessibility metrics
Correlation Analysis: Discovered that properties within 500m of high-crime areas showed 12-18% lower valuations. Single-family dwellings in low-crime zones appreciated 7% faster than city average.

Technical Implementation

Technology Stack

  • SQL: PostgreSQL for relational database management, complex joins, and aggregations
  • Python: pandas, NumPy, scikit-learn, matplotlib, seaborn for data processing and analysis
  • Tableau: Interactive dashboards for executive presentations and stakeholder reporting
  • Excel: Quick analyses, pivot tables, and client-facing reports
  • Git: Version control for code and data pipeline management

Data Architecture

Designed a scalable data warehouse architecture with staging, transformation, and presentation layers. Implemented automated ETL pipelines with error handling, logging, and data quality checks. Optimized query performance for real-time dashboard updates.

Visualization Strategy

  • Executive dashboards with high-level KPIs and trends
  • Operational dashboards for day-to-day monitoring
  • Analytical deep-dives with drill-down capabilities
  • Geographic heat maps for spatial pattern visualization
  • Time-series charts for temporal trend analysis

Key Findings & Insights

Crime Trends

  • 41% crime reduction in areas with increased police presence during peak hours (12:00-18:00)
  • Vehicle-related crimes showed seasonal patterns with 30% increase during summer months
  • Identity theft incidents increased 45% year-over-year, indicating emerging threat
  • Street crimes concentrated in commercial districts; residential areas showed different patterns

Sales Patterns

  • Strong correlation between low-crime neighborhoods and premium product sales
  • Sales velocity 25% higher in areas with crime rates below city average
  • Customer retention rates 18% better in safer neighborhoods
  • Seasonal sales peaks aligned with post-holiday periods (January-February)

Property Market Dynamics

  • Properties near high-crime locations (800 N Alameda) depreciated 3-5% annually
  • Single-family homes in low-crime areas commanded 15% premium
  • Multi-unit dwellings showed more resilience to neighborhood crime fluctuations
  • Properties near frequent crime locations (The Grove, Figueroa St) had longer time-on-market
